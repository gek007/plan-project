Custom Subagent Ideas
Used for domain specific tasks

User Experience (UX) Reviewer
Analyzes user flows, identifies UX pain points, suggests interface improvements, reviews accessibility compliance, and ensures consistent design patterns across the application. Focuses on user journey optimization and interface usability.

API Designer
Reviews API endpoints for RESTful best practices, ensures consistent response formats, validates request/response schemas, suggests performance optimizations, and maintains API documentation standards.

Security Reviewer
Audits code for security vulnerabilities, reviews authentication / authorization implementations, checks for data exposure risks, validates input sanitization, and ensures compliance with security best practices.

Test Runner
Executes test suites, analyzes test coverage, identifies failing tests, suggests new test cases for uncovered code paths, and ensures CI/CD pipeline health.

Database Admin
Manages database schema migrations, optimizes query performance, reviews data modeling decisions, ensures proper indexing strategies, and monitors database health and security.
